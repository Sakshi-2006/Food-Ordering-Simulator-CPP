# Food-Ordering-Simulator-CPP
A console-based food ordering system in C++ with menu-driven interface, quantity handling, and GST-based billing.

## Features

* Menu-driven interface
* Take user input
* Multiple item ordering
* Bill calculation with GST
* Quantity selection and support
* Order summary

## Concepts used:
- Loops (do-while)
- Switch-case
- Variables & basic logic

## Technologies Used

* C++
* Standard Input/Output (cin/cout)

## How to Run

* Compile using:
g++ main.cpp -o app
* Run using:
./app
